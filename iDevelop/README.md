# iDevelop

Codex Dashboard 系の companion project。

- 規約入口: `iDevelop/docs/index.md`
- 実開発入口: `iDevelop/develop/index.md`
- companion project: `../iSensorium`

## Entry Command

- dashboard 起動: `npm run dashboard`
- build 済み成果物の preview: `npm run preview`
- Windows 初心者向けマニュアル: `docs/process/UX_check_work_flow.md`

`npm run dashboard` を実行すれば、ローカルで dashboard を開いて UI 操作しながら確認できる前提で開発を進める。
